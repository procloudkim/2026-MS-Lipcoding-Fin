## 30초 요약 (엘리베이터 피치)

- **문제**: 할 일이 여러 곳에 흩어져 있을 때, 사람은 "무엇부터 할지"를 **결정**하는 데 에너지를 쓴다. 이 "결정 비용"이 진짜 생산성 손실이다.
- **해법**: 단순 To-Do 정리가 아니라, **결정을 대행**한다. 각 항목에 `지금 바로 / 오늘 안에 / 미루기 / 위임 / 버리기` 판정을 부여하고, **입력한 순서가 아니라** 마감·영향·시간 민감도로 재정렬한 뒤 *왜 이 순서인지*까지 설명한다.
- **차별점(투명성 UX)**: "내가 1번으로 적어서 1번이 된 것 아니냐"는 불신을 없애기 위해, 응답에 `orderRationale`(재정렬 근거)을 담아 화면 상단에 항상 노출한다.
- **신뢰성(정직한 폴백)**: AI 런타임이 불가한 환경에서도 동일 스키마의 결정적(deterministic) 결과를 HTTP 200으로 반환하고, 화면 배지로 `Copilot SDK` vs `오프라인 결정 엔진`을 **정직하게** 구분 표기한다.

| 지표 | 값 |
| --- | --- |
